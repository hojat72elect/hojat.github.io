Notify Plus

Notify Plus is a feature-rich tool box that makes your Android experience even more fun! The app provides you with 2 main tools:

  1. A sticky notification containing 8 modifiable shortcuts to user's favorite apps.
  2. A floating widget which can work as an equivalent of floating control center in iOS.

Apart from all the fun, the most important experiences I took from this project were these:

  1. Google Play Store publishing process.
  2. Git Version Control.
  3. RemoteViews in Android.
  4. Sticky notifications in Android.
  5. GUI control via services.
  6. Usage of 3rd party libraries via Gradle dependencies.
